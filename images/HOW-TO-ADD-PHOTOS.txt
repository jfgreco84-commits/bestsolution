==========================================================
  BEST SOLUTION SITE - HOW TO ADD / CHANGE PHOTOS YOURSELF
==========================================================

No coding needed. You just drop correctly-NAMED .jpg photos into
THIS "images" folder. The website automatically shows them.

----------------------------------------------------------
THE RULES
----------------------------------------------------------
1. Photo must be a .jpg file (JPG). If yours is .png, open it in
   Paint, "Save As" JPEG, done.
2. Name the file EXACTLY as shown below (lowercase, no spaces).
3. Drop it in this folder. If Windows asks to replace, say YES.
4. Refresh the web page to see it. (To make it LIVE for customers,
   the file also has to be uploaded to GitHub - ask Claude to push,
   or upload on github.com. See bottom.)

A photo only shows up if the name matches. Wrong name = nothing
breaks, it just won't appear.

TO MAKE CHANGES LIVE FOR CUSTOMERS
----------------------------------------------------------
Editing files here only changes YOUR computer's preview. To update
the real site (jfgreco84-commits.github.io/bestsolution):
